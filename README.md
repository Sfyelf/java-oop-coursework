# Java OOP Coursework

## Overview

This repository contains Java exercises and mini-projects created while learning Object-Oriented Programming.

The goal of this repository is to practice core OOP principles and apply them through small programming assignments and examples.

## Topics Covered

- Classes and objects
- Constructors
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Interfaces
- Method overriding
- Method overloading
- Exception handling
- Basic file handling

## Tech Stack

- Java
- Object-Oriented Programming
- IntelliJ IDEA / Eclipse / Visual Studio Code
- JDK
